Disaat saya melakukan penetrations testing terhadap website anda, saya menemukan bug Bypass Admin di website anda.

Bug Bypass Admin dapat memungkinkan penyerang untuk memasukkan query SQL secara paksa pada form login yang nantinya akan dibaca oleh komputer sebagai sebuah instruksi login (karena instruksinya memaksa, jadi tanpa user & pass pun komputer akan memprosesnya).
