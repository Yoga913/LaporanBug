Disaat saya melakukan penetrations testing terhadap website anda, saya menemukan bug CSRF (Cross Site Request Forgery) di website anda.

Bug CSRF (Cross Site Request Forgery) dapat memungkinkan penyerang untuk mengirimkan link atau halaman berisi request tersembunyi pada pengguna (korban), yang dieksekusi oleh pengguna tersebut ke website target.
